# 📝 MERN To-Do App

A full-stack **To-Do Management Application** built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js). The application allows users to securely manage their daily tasks with authentication, task tracking, filtering, and an interactive dashboard.

---

## 🚀 Live Demo

🌐 **Live Application:** todolist-lriy.vercel.app

---

## 🧰 Technologies Used

### 💻 Frontend

* React.js (Vite)
* React Router DOM
* Axios
* React Toastify
* Material UI Icons
* CSS3

### 🌐 Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt.js
* CORS

---

## ✨ Features

* 🔐 User Registration & Login using JWT Authentication
* 👤 Protected Routes
* ➕ Create New Tasks
* ✏️ Edit Existing Tasks
* ❌ Delete Tasks
* 📋 Personal "My Tasks" Page
* 🌍 Dashboard displaying all users' tasks
* 🔍 Search Tasks by Title
* 📂 Filter Tasks by Status (All / Pending / Completed)
* ✅ Toggle Task Status (Pending ↔ Completed)
* 📊 Dashboard Statistics

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
  * Completion Percentage
* 📅 Task Creation Date Display
* 🎨 Clean Responsive UI
* 🔔 Toast Notifications for Success & Error Messages
* 🏷️ Task Priority Support (Low / Medium / High)

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/Yuktamahajan1999/Todolist.git
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Todo-List
npm install
npm run dev
```
